# SAS

This project is a React web application built with Vite and TypeScript.

## Getting Started

1. **Install dependencies**
	```sh
	npm install
	```

2. **Start the development server**
	```sh
	npm run dev
	```
	The app will be available at [http://localhost:5173](http://localhost:5173) by default.

3. **Build for production**
	```sh
	npm run build
	```

4. **Preview the production build**
	```sh
	npm run preview
	```

## Project Structure
- `src/` — Main source code (React components, styles, types, data)
- `public/` — Static assets (if needed)
- `index.html` — Main HTML file
- `package.json` — Project configuration and scripts

## Technologies Used
- React
- TypeScript
- Vite
- Tailwind CSS
- Node.js (for tooling)

## Notes
- No Bolt AI or React Native libraries are used.
- For icons, `lucide-react` is included.
- All configuration files are set for a standard React + Node.js workflow.
